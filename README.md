# 小区物业管理系统

## 项目简介
小区物业管理系统是基于Java Web技术, Springboot框架,Mysql数据库的项目。分为以下几个模块：1.用户登录注册  2.房屋管理 3.车位管理 4.缴费系统 5.社区服务 6.个人中心等。 适合作为毕业设计项目、大作业、课程设计等

Java;SpringBoot;Html/Css、JavaScript;Mysql数据库
